# 2025-1MB-cidadeconectada-G03

Introdução

O site Onisciente é uma aplicação voltada à usuários de transporte público, em específico, os ônibus de Porto Alegre. Ele foi criado para ajudar quem faz uso do sistema de transporte público e simplificar cada vez mais a sua rotina. A ideia foi concebida após diversas reflexões pela equipe sobre como facilitar todo o ato de pegar um ônibus.  Suas principais ferramentas são: o gerador de rotas e o tutorial de como andar de ônibus.
